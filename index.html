about

    

      
I work on how machines represent where something is. Most of my research sits at the intersection of geospatial data and representation learning — turning coordinates, satellite imagery, and demographic signals into embeddings that language and vision models can actually reason over.

      
At Esri, I'm building the Global Location Encoder, a dual-encoder model pairing DINOv2 and Terramind to learn representations for any point on the globe, benchmarked against models like GeoCLIP and SatCLIP. Alongside that: tooling that gets these models running inside ArcGIS Pro, and LLM-assisted pipelines that turn raw geospatial data into something queryable.

selected work


    

      

        Esri
        Geospatial AI Researcher · current
      


      

        
Global Location Encoder

        
Dual-encoder model (DINOv2 + Terramind) producing global-scale location embeddings from Sentinel-2 imagery, with weighted grid-based sampling and a relational loss. Benchmarked against GeoCLIP and SatCLIP.

      


      

        
High-Resolution US Location Embeddings

        
A fine-grained location encoder tuned for the continental US, extending the global model to finer spatial resolution for downstream prediction tasks.

      


      

        
Multimodal Demographic Model

        
Combines tabular, image, text, and coordinate signals through cross-attention fusion to predict demographic outcomes at the grid-cell level — outperforming Google's PDFM on downstream benchmarks.

      


      

        
LLM-Assisted Caption Generation Pipeline

        
A two-stage LLM pipeline (running against Ollama / vLLM) that generates natural-language demographic captions for roughly 33,000 US ZIP codes from Census and ACS data.

QueryEarth

        
A DAG-based geospatial query pipeline built on multi-resolution vision embeddings, with an LLM router that converts natural-language queries into ordered pipeline steps.

        
python · llm-routing · geospatial

      


      

        
ArcGIS Pro DLPK Tools

        
Deep learning package tools for ArcGIS Pro — a unified VLM encoder (OpenCLIP, auto-detecting image vs. text) and a SkyCLIP embeddings tool merging raster-function and classifier interfaces.

        
python · openclip · arcgis

      


      

        
autoresearch

        
An autonomous architecture-search system: a local LLM iteratively edits a training script and evaluates spatial embeddings on downstream F1 across health, climate, FEMA, and SVI datasets via an untouched eval harness.

        
python · optuna · local-llm

      


      

        
eval_embeddings & HPO repos

        
An evaluation pipeline for spatial embeddings (R² scoring, k-fold cross-validation) alongside two HPO repos using Optuna NSGA-II to search autoencoder architectures for geospatial feature encoding.

        
pytorch · optuna · scikit-learn
